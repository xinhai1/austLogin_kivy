# austlogin_kivy

### 简介

用 `python3` 编写的桌面程序, 用于登录安理大校园网, 使用了 `kivy` 模块

使用 `pyinstaller` 打包exe

### 按钮功能

`登入`: 登录校园网

`登出`: 登出校园网

`保存`: 保存你的配置                     

`清空`: 清空你已保存和填写的配置
